# Scan background animations

Drop your animation images here (PNG or WebP recommended). They render behind scan UIs at low opacity (~7–9%).

| File | Used on |
|------|---------|
| `url-scan.png` | Home URL scanner |
| `email-scan.png` | Home email checker |
| `sandbox-scan.png` | Sandboxing page |
| `agentic-scan.png` | Agentic page |

If a file is missing, CSS animated gradients are used instead.
